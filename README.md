# 🎯 Система достижения целей 2026

Личный трекер целей с еженедельным планированием. Построен как PWA — устанавливается на телефон как приложение.

## Возможности

- **Dashboard** — обзор прогресса по всем ролям
- **Еженедельный план** — задачи по каждой роли на текущую неделю
- **Трекер целей** — слайдеры прогресса и заметки по каждой цели
- **Еженедельный обзор** — рефлексия и история за прошлые недели
- **Офлайн-режим** — работает без интернета (PWA)
- **Установка на телефон** — открой в Chrome → «Добавить на главный экран»

## Установка на Android

1. Открой ссылку сайта в **Chrome** на Android
2. Нажми меню `⋮` → **«Добавить на главный экран»**
3. Приложение установится как нативное — работает офлайн

## Технологии

- Vanilla JS, HTML, Tailwind CSS
- localStorage для хранения данных
- Service Worker для офлайн-режима
- PWA manifest для установки на устройства

## Развитие

Данные хранятся локально в браузере (`localStorage`). При необходимости синхронизации между устройствами — планируется добавить облачный бэкенд.
